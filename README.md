## ✨ Overview

**Liquid Obsidian Studio** is a high-performance, AI-powered web development environment designed for the modern era. It combines the tactile elegance of **Neumorphic Design** with the raw power of **Gemini 1.5 Pro** to help you build, preview, and deploy websites in record time.

> "The bridge between imagination and production-ready code."

---

## 🌈 Features at a Glance

### 🤖 AI Website Agent
Describe your vision in plain English. Our agent handles the complex HTML, CSS, and JS, allowing you to focus on the creative direction.

### 🎨 Liquid Obsidian Theme
A stunning, high-density dark neumorphic UI. Every button, card, and input is crafted with depth and precision.

### ⚡ Real-Time Compilation
Watch your code transform instantly. With 60FPS live preview and multi-device toggles, what you see is exactly what you get.

### 📦 Asset Management
Need FontAwesome? Google Fonts? GSAP? Just drop the CDN link in the Assets manager and you're ready to go.

## 📥 Getting Started

### 🔑 Prerequisites
- Node.js 18+
- A Gemini API Key from [Google AI Studio](https://aistudio.google.com/)

### 🚀 Quick Start
```bash
# Clone the studio
git clone https://github.com/your-username/liquid-obsidian-studio.git

# Enter the studio
cd liquid-obsidian-studio

# Install the engine
npm install

# Ignite the server
npm run dev
```
